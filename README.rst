*********************
Contributing
*********************

**Creating the documentation**

Dependencies:

    * Python
    * `Sphinx <http://sphinx.pocoo.org>`_ 
    * Git

`pip install -r requirements.txt`
    

**Clone this repo:**
::

    git clone https://github.com/okfn-brasil/querido-diario-doc.git

**Make documentation:**
::

    make html

Open file `build/html/index.html` in your browser.
